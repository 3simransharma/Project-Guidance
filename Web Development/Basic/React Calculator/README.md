# React_Calculator
a simple calculator made using React framework
# Functionalities
- Addition 
- Multiplication
- Division
- Percentage
- Decimal use
- Subtraction
- Reset
Easy to use and run.
# Installation
- Just type `npm install` in the folder terminal where the project is kept. 
- Now type `npm start`.
- Enjoy
# Build with
- Html
- Css
- Javscript
- React Framework
